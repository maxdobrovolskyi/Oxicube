# Oxicube

2D voxel engine written in Rust with wgpu.

## What is this

A from-scratch engine for rendering tile-based voxel worlds. Think somewhere between Factorio and Terraria but with RPG elements baked in.

Still very early — working on the rendering pipeline right now.

## Tech

- Rust
- wgpu
- 16x16 pixel tiles

## Building

```
cargo build --release
cargo run
```

## Status

WIP. Nothing playable yet.
